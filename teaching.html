Teaching

I’m an Undergraduate Teaching Assistant for ECE 2k7 (Electrical Engineering Fundamentals Laboratory). The work is hands-on: building circuits, taking measurements, and debugging when reality disagrees with the schematic.

ECE 2k7 (TA)

  • Helping students build and debug circuits
  • Guiding lab experiments with oscilloscopes and instrumentation
  • Teaching systematic debugging habits (hypothesis → test → iterate)

Tools Used in Lab

  • oscilloscope
  • multimeter
  • function generator
  • breadboards
  • logic ICs

Debugging loop

observe → hypothesize → test → iterate

Verify power/ground and continuity first. Then isolate one stage at a time. If a waveform looks wrong, confirm probe grounding and reference points before changing the circuit.
